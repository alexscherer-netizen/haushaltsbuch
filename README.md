# 💰 Haushaltsbuch

Modernes Haushaltsbuch für Paare – läuft direkt im Browser, keine Installation nötig.

## 🚀 Live öffnen

👉 **[haushaltsbuch öffnen](https://DEIN-USERNAME.github.io/haushaltsbuch/)**

> `DEIN-USERNAME` durch deinen GitHub-Benutzernamen ersetzen.

## ✨ Features

- Zwei Personen mit individuellen Namen, Einnahmen & Ausgaben
- Gemeinschaftskonto mit automatischer Saldoberechnung
- Einkauf-Tab: monatliche Einkäufe individuell erfassen
- Sonderausgaben: jährliche & vierteljährliche Ausgaben mit Rücklageberechnung
- Monats- & Jahresübersicht mit Tabellen
- Diagramme (Balken + Donut)
- Dark Mode, Export/Import als JSON
- Daten werden automatisch lokal im Browser gespeichert
